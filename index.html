What is Walrus Memory?

Walrus Memory gives AI assistants like Claude and Gemini long-term memory — they remember what you tell them even after you close the app and start a new conversation. Claude and Gemini share the same memory, so what one knows, the other knows too.

3 Connect Walrus Memory to Gemini CLI

📌 Once set up, Claude and Gemini automatically share the same memories — no extra configuration needed.
💬 Ask Claude to do all of Part 3 for you
Set up Gemini CLI on my Mac with Walrus Memory: install @google/gemini-cli globally, create the folder ~/Documents/gemini-workspace, and configure ~/.gemini/settings.json to use the same Walrus Memory MCP server that you use (npx -y @mysten-incubation/memwal-mcp). Then tell me how to authenticate Gemini for the first time.
Open Claude ↗
5
Install Gemini CLI and configure Walrus Memory
✓ Done
›

Install Gemini CLI:

npm install -g @google/gemini-cli

Create a dedicated workspace folder (prevents Mac security alerts):

mkdir -p ~/Documents/gemini-workspace

Open or create ~/.gemini/settings.json and add:

{
  "sandbox": true,
  "mcpServers": {
    "memwal": {
      "command": "npx",
      "args": ["-y", "@mysten-incubation/memwal-mcp"]
    }
  }
}

Authenticate and launch Gemini from the workspace folder:

cd ~/Documents/gemini-workspace && gemini

Your browser will open — sign in with Google.

Verify Gemini can read your memories:

💬 Type this inside Gemini CLI
Do you have access to Walrus Memory tools? Call memwal_recall and search for where do I work.
✅ Gemini should find the same memory you saved from Claude.

Error "426 Upgrade Required" Ask Claude: "Update the Walrus Memory SDK to the latest version."

∿ How It Works

Claude and Gemini connect to the same local server, which stores memories on Walrus — a decentralized storage network built by Mysten Labs — with end-to-end encryption. Only you can read your memories.

Claude (Cowork) ──┐ ├──▶ Walrus Memory MCP (runs on your Mac) Gemini CLI ──┘ │ ▼ MemWal Relayer (relayer.memwal.ai) │ ▼ Walrus — decentralized encrypted storage │ Sui blockchain — access control
Walrus Memory is in beta — free to use. For the latest docs visit docs.memwal.ai
